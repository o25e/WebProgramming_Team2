HTML은 Hyper Text Markup Language의 약어이다. 여기서 hypertext란 현재 페이지에서 다른 페이지로 이동할 수 있는 텍스트를 의미한다. 그리고 markup language는 문서의 구조를 정의하는 언어이다. 즉, HTML은 웹 페이지의 구조를 만드는 언어이다 HTML은 태그가 달린 요소(element)들로 이루어져 있다. 태그를 사용하여 요소를 만드는 방법은 다음과 같다. <태그명>내용</태그명> 태그에 따라 다음과 같은 경우도 있다. <태그명> 이렇게 태그 하나로 구성된 요소들을 void elements, 빈요소라고 한다. HTML 문서는 모두 태그가 달린 요소로 이루어져 있다. 그리고 요소 안에 속성(attribute)을 넣을 수 있다. 넣는 방법은 다음과 같다. 속성을 넣어서 요소에 원하는 대로 변화를 줄 수 있다. HTML로 웹 페이지를 만드려면, 반드시 html, head, body 3가지의 태그를 사용해야 한다. 사용 형태는 다음과 같다. 
<html>
    <head>
        문서의 기본 정보를 작성한다.
    </head>
    <body>
        문서 화면에서 보이는 모든 내용을 작성한다.
    </body>
</html>
html 태그는 html 문서의 시작과 끝을 알린다. head 태그는 문서의 기본 정보를 담고 있다. body 태그는 문서의 보이는 모든 내용을 담고 있다. 그리고 html파일 앞부분에 문서 유형을 다음과 같이 선언해야 한다. <!DOCTYPE html> 이는 문서가 html5임을 알려주는 선언문이다. html5는 html의 최신 버전이다. 다음은 지금까지 배운 것을 적용한 html의 기본 구조이다. 
<!DOCTYPE html>
<html>
    <head>
        문서의 기본 정보를 작성한다.
    </head>
    <body>
        문서 화면에서 보이는 모든 내용을 작성한다.
    </body>
</html>
