입력 양식 태그를 통해 사용자로부터 정보를 입력 받을 수 있다. <form> 태그로 입력 양식 영역을 생성할 수 있다. 다음과 같이 쓴다. 
<!--form 태그-->
<!DOCTYPE html>
<html>

<head>
    <title>form 태그</title>
</head>

<body>
    <form>
        <input type="text" name="search">
        <input type="submit">
    </form>
</body>

</html>
form 태그는 action 속성으로 데이터를 전송할 위치를 정할 수 있고 method 속성으로 전송방식을 정할 수 있다. method=”get”은 주소에 데이터를 입력해서 전달한다.(주소가 바뀐다.) method=”post”는 주소 변경 없이 다른 방법으로 전달한다. 입력 양식 태그의 종류는 다음과 같다. 그 중 form태그와 input태그를 사용한 예시를 살펴보자. 
<!--입력 양식 태그-->
<!DOCTYPE html>
<html>
    <head>
        <title>입력 양식 태그</title>
    </head>
    <body>
        <form action="">
            <!--사용자가 입력하는 입력 양식-->
            <input type="text" name="text" value="text"><br>
            <input type="password" name="password" value="password"><br>
            <input type="file" name="file" value="file"><br>
            <input type="checkbox" name="checkbox" value="checkbox"><br>
            <input type="radio" name="wow" value="radio">
            <input type="radio" name="wow" value="radio"><br>
            <!--보이지 않는 입력 양식-->
            <input type="hidden" name="hidden" value="hidden"><br>
            <!--버튼-->
            <input type="button" value="button"><br>
            <input type="reset" value="reset"><br>
            <input type="submit" value="submit"><br>
            <input type="image" src="http://place-hold.it/100x200">
        </form>
        <br>
        <form method="post">
            <table>
                <tr>
                    <td><label for="username">이름</label></td>
                    <td><input id="username" type="text" name="username"></td>
                </tr>
                <tr>
                    <td>성별</td>
                    <td>
                        <input id="man" type="radio" name="gender" value="m">
                        <label for="man">남자</label>
                        <input id="woman" type="radio" name="gender" value="w">
                        <label for="woman">여자</label>
                    </td>
                </tr>
            </table>
            <input type="submit" value="가입">
        </form>       
    </body>
</html>
참고로 “http://place-hold.it/” 사이트는 뒤에 원하는 크기를 입력하면 크기가 바뀐다. “http://place-hold.it/가로x세로”로 입력하면 된다. 첫번째 form 안에 여러 종류의 input 태그가 있는데 type 속성에 따라 모양과 특성이 바뀐다. 그리고 value 속성으로 내부에 글자를 입력해 둘 수 있다. 두번째 form 태그는 method=”post”로 하여 데이터를 주소에 담지 않고 전달하였다. method 속성을 입력하지 않으면 기본적으로 “get”으로 설정된다. 그리고 radio 태그는 name 속성의 값을 같게 해야 여러 항목 중 하나만 선택된다. label 태그의 for 속성에는 연결하고 싶은 input 요소의 id를 넣는다. 그러면 해당 id를 가진 요소와 연결되어 label의 텍스트를 클릭했을 때 연결된 요소에 포커싱이 된다.
