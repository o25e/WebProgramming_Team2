이번 장에서는 글자 태그 중 하나인 앵커 태그를 알아보자. 앵커 태그는 a 태그라고도 하며 글자에 하이퍼링크를 달아주는 역할을 한다. 다음과 같이 생겼다. 하이퍼링크를 달기 위해서는 href 속성에 이동할 경로를 입력한다. 경로에는 4가지가 있는데 다음과 같다. 절대 경로 절대 경로는 기준이 최상위 위치로 정해져 있다. 예시로 네이버 메인 페이지가 있다. http://naver.com 절대 경로에서 /animal.jpg는 현재 웹 사이트 최상위 위치의 animal.jpg 파일을 뜻한다. 상대 경로 상대 경로는 현재 웹 페이지를 기준으로한다. 사용 예시는 다음과 같다. animal.jpg는 웹 페이지가 있는 폴더의 animal.jpg 파일이다. image/animal.jpg는 웹 페이지가 있는 폴더에 포함된 image 폴더의 animal.jpg 파일이다. ../animal.jpg는 웹 페이지가 있는 폴더의 상위 폴더에 있는 animal.jpg 파일이다. 아이디 경로 여기서 아이디(id)란 태그에 다는 속성 중 하나이다. id 속성을 사용하면 태그에 원하는 id를 지어줄 수 있다. 아이디 경로는 아이디 앞에 ‘#’을 붙여 작성한다. 예시는 다음과 같다. #name는 id 속성이 name인 태그의 위치로 이동시켜준다. 즉, 문서 내부에서도 이동할 수 있다. 메일경로 mailto:hanb@hanbit.co.kr은 해당 주소로 메일을 전송해준다. 예제를 살펴보자. 다음은 절대경로 앵커태그 예시이다. 
<!--절대경로로 웹페이지 연결하기-->

<!DOCTYPE html>
<html>
    <head>
        <title>anchor</title>
    </head>
    <body>
        <a href="http://www.hanbit.co.kr">한빛미디어</a>
        <br>
        <a href="http://www.naver.com/">네이버</a>
        <br>
        <a href="http://www.daum.com/">다음</a>
    </body>
</html>
다음은 아이디 경로 예시이다. 
<!--아이디경로로 내부에 연결하기-->
<!DOCTYPE html>
<html>
    <head>
        <title>anchor</title>
    </head>
    <body>
        <a href="#alpha">alpha 부분</a>
        <a href="#beta">beta 부분</a>
        <hr>
        <h1 id="alpha">Alpha</h1>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
        <div></div>
        <h1 id="beta">Beta</h1>
        <p>Vivamus elementum pretium tortor, in tincidunt mauris gravida et.</p>
    </body>
</html>
위 예시는 문서가 짧아 하이퍼링크를 클릭해도 이동효과가 나타나지 않지만 긴 문서에서는 제대로 이동한다.
