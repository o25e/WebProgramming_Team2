반응 선택자란?

반응 선택자는 사용자의 행동에 따라 스타일이 변하도록 하는 CSS 선택자이다. 주로 :hover, :active, :focus, :visited 등을 사용해서, 마우스를 올렸을 때, 클릭했을 때, 링크를 방문했을 때 등을 처리할 수 있다다. 

a:hover {
        color: red;
        text-decoration: underline;
      }

반응 선택자 예시

아래 버튼을 마우스로 올리면 색상이 변경된다. (hover 효과). 링크를 클릭하면 색상도 달라진다.
