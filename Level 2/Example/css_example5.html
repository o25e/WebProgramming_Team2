속성 선택자 (Attribute Selector)

속성 선택자는 HTML 태그에 있는 특정 속성(attribute)을 기준으로 스타일을 적용할 수 있게 해주는 CSS 선택자다. 예를 들어, 특정 type, href, data-* 속성을 가진 요소에만 스타일을 줄 수 있다.

  • [속성] → 해당 속성이 존재하면 선택
    예: [disabled] → disabled 속성이 있는 요소
  • [속성=값] → 해당 속성과 정확한 값을 가진 요소
    예: [type="text"] → <input type="text">
  • [속성~=값] → 공백으로 구분된 여러 값 중 하나가 일치할 경우
  • [속성^=값] → 값이 지정한 문자열로 시작할 경우
  • [속성$=값] → 값이 지정한 문자열로 끝날 경우
  • [속성*=값] → 값이 지정한 문자열을 포함할 경우

➡ 이렇게 하면 클래스를 쓰지 않아도 속성 기준으로 스타일을 줄 수 있다.

글자 굵기 설정

글자의 두께(굵기)를 조절하는 속성이다.

  • normal → 기본 굵기 (400)
  • bold → 굵게 (700)
  • lighter / bolder → 부모 요소보다 더 얇거나 굵게
  • 100 ~ 900까지 100단위로 사용 → 숫자가 클수록 더 굵게

예:

  • font-weight: 100; → 매우 얇음
  • font-weight: 400; → 일반 굵기
  • font-weight: 700; → 굵게
  • font-weight: 900; → 매우 굵게
