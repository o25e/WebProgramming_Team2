요소 외부, 내부 여백 설정이란?

요소의 외부와 내부 여백을 설정하는 CSS 속성은 margin과 padding이다. 이 두 속성은 요소의 크기와 배치에 중요한 역할을 한다.

외부 여백 (margin)

margin은 요소의 외부 공간을 설정한다. 이 속성은 요소와 다른 요소 사이의 간격을 결정한다. 

          
    .box {
         margin: 20px; /* 외부 여백 20px 설정 */
     }

내부 여백 (padding)

padding은 요소의 내부 공간을 설정한다. 이 속성은 요소의 콘텐츠와 테두리 사이의 간격을 조정한한다. 

          
    .box {
        padding: 30px; /* 내부 여백 30px 설정 */
    }
