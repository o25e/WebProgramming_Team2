클래스 선택자란?

클래스 선택자는 HTML 요소에 class 속성을 주고, CSS에서 그 이름 앞에 마침표(.)를 붙여서 선택한다. 여러 요소에 같은 스타일을 적용할 때 유용하다. 한 요소에 여러 클래스를 띄어쓰기로 같이 쓸 수도 있다.

  • 예: <p class="text">내용</p>
  • CSS: .text { color: blue; }
  • 여러 개 클래스: <div class="box shadow">

글꼴 종류 설정

글꼴을 지정할 때는 font-family 속성을 사용한다. 하나의 글꼴이 없을 경우를 대비해서 여러 개를 쉼표로 나열하고, 마지막엔 일반적인 계열(serif, sans-serif 등)을 넣는다.

  • '돋움', '굴림', 'Arial', 'sans-serif' → 앞에서부터 가능한 글꼴을 적용한다
  • 앞에 있는 글꼴부터 순서대로 시도해서, 시스템에 없는 경우 뒤에 있는 걸 사용한다
  • 글꼴 이름에 띄어쓰기가 있을 경우 ' ' (작은 따옴표)로 감싼다

예:

  • font-family: 'Noto Sans KR', 'Malgun Gothic', sans-serif;
  • font-family: 'Georgia', 'Times New Roman', serif;
