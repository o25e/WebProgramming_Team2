CSS란?

CSS는 Cascading Style Sheets의 약자로, HTML 문서의 디자인과 레이아웃을 담당하는 스타일 언어이다.

CSS 작성 방법

  1. 인라인 스타일 (HTML 요소에 직접 작성)
    2. <p style="color: red;">이 문장은 빨간색입니다.</p>
  2. 내부 스타일시트 (<style> 태그 사용)
    3. <head>
        <style>
          p {
            color: red;
          }
        </style>
      </head>
  3. 외부 스타일시트 (별도 CSS 파일 연결)

    4. html

    <link rel="stylesheet" href="style.css">

    style.css

    p {
        color: red;
      }

CSS 기본 구조

선택자 { 속성: 값; } 형식으로 작성하며, 선택자는 HTML 요소를 선택해 스타일을 적용하는 역할을 한다.

전체 선택자

전체 선택자(*)는 HTML 문서의 모든 요소에 동일한 스타일을 적용한다다. 

* {
        속성: 값;
      }

글자 색상 지정 방법

글자 색상은 color 속성을 사용해서 설정할 수 있다. 방법은 여러 가지가 있다:

  1. 이름으로 지정
    2. color: red;
color: blue;
  2. 16진수 코드로 지정
    3. color: #ff0000;
color: #00ff00;
  3. RGB 함수로 지정
    4. color: rgb(255, 0, 0);
color: rgb(0, 255, 0);
