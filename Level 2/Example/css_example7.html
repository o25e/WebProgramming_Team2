자손 선택자란?

자손 선택자는 특정 요소 내부의 모든 하위 요소를 선택해서 스타일을 적용할 수 있게 해준다. 띄어쓰기를 통해 표현하며, 형식은 A B {속성:값;}이다. 

.parent span {
        color: #fd8181;
      }

이 문장에서 핑크 글씨는 <span> 태그이고, .parent span 선택자로 스타일이 적용된다.

텍스트 꾸밈 (text-decoration)

텍스트 꾸밈은 text-decoration 속성으로 설정하며, 텍스트에 밑줄, 취소선, 윗줄 등을 넣을 수 있다. 

.decoration {
        text-decoration: underline;
      }

이 문장은 밑줄이 그어져 있어요.

이 문장은 취소선이 그어져 있어요.

이 문장은 윗줄이 있어요.

이 문장은 텍스트 꾸밈이 없어요 (기본 상태).

주로 링크 태그(<a>)의 밑줄 제거에도 많이 사용된다. 

a { text-decoration: none; }
