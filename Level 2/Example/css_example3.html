아이디 선택자란?

아이디 선택자는 HTML 요소에 부여된 고유한 id 속성을 기준으로 스타일을 적용하는 방법이다. 선택자 앞에 #을 붙여서 사용하며, 한 HTML 문서 내에서 같은 id를 두 번 이상 사용하면 안 된다. 

<p id="highlight">이 문장은 핑크크 배경과 흰 글씨로 강조됩니다.</p>

#highlight {
  color: white;
  background-color: #fd8181;
  font-size: 18px;
}

이 문장은 핑크 배경과 흰 글씨로 강조됩니다.

글자 크기 단위

CSS에서 글자 크기를 지정할 땐 다양한 단위를 사용할 수 있다. 각각의 단위는 기준과 용도가 다르기 때문에 상황에 맞게 선택하는 게 중요하다.

단위 의미 예시
px 고정 크기 (화면에 딱 맞는 정해진 크기) font-size: 20px;
em 부모 요소의 글꼴 크기를 기준으로 배수 font-size: 1.5em;
rem html 요소(root)의 글꼴 크기를 기준으로 배수 font-size: 1.2rem;
% 부모 요소의 크기를 기준으로 백분율 font-size: 120%;

팁: 반응형 디자인을 하거나, 사용자가 글꼴 크기를 조절할 수 있게 하려면 em이나 rem을 사용하는 게 좋다.
